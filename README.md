# DPM Article for GitHub Pages

This folder is a simple GitHub Pages article site.

## Files

- `index.html` is the article page.
- `style.css` controls the design.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html` and `style.css` from this folder.
3. Go to the repository settings.
4. Open **Pages**.
5. Set the source to the `main` branch and `/root`.
6. Save.

GitHub will give you a link like:

```text
https://your-username.github.io/your-repository-name/
```
